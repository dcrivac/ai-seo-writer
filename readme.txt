A full-suite content creation platform. Generate, refine, and perfect your articles with a powerful set of AI tools, all within a unique, dark-mode interface.

== Description ==

Tired of the same boring AI tools? AI SEO Writer breaks the mold. It's more than just a content generator; it's a complete workflow.

Start by defining your article's topic, tone, and audience. Watch in real-time as the AI crafts your post. Then, use the integrated "Tijuana Tune-Up" panel to instantly generate SEO meta descriptions, suggest tags, create social media teasers, and find featured images.

This is the ultimate tool for streamlining your content creation process, all wrapped in a custom dark-mode interface that stands out.

= Features =

* **Unique Dark-Mode UI:** A custom-designed, dark interface with vibrant accents.
* **Advanced Content Controls:** Define the article's tone and target audience before generation.
* **Live Progress Indicator:** See the AI's creation process unfold in real-time.
* **"Tijuana Tune-Up" Panel:** A post-generation suite of tools to perfect your article.
    * One-click SEO Meta Description generation.
    * AI-powered SEO Tag suggestions.
    * Instant Social Media Teaser creation for X/Twitter.
    * Quick link to find royalty-free featured images.
* **Automatic Draft Creation:** Generated articles are saved as drafts, ready for your review.

== Installation ==

1.  Upload the `ai-seo-writer` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to the **AI Writer > Settings** page and enter your OpenAI API Key.
4.  Navigate to the **AI Writer** page and start creating!

== Frequently Asked Questions ==

= Do I need an OpenAI account? =

Yes, this plugin requires an API key from OpenAI. You can get one by signing up on their website.
